# MVP Base Asset Pack

This pack is fully custom and generated by `pnpm run build:assets`.

Generated files:

- `scripts/mvp.shader`
- `textures/mvp/*.tga`
- `mapsrc/mvp_box.map`

`mapsrc/mvp_box.map` is source only.
Compile it with:

```bash
pnpm run build:map
```

This uses `q3map2` from `PATH` by default.
Override binary path with `STRAFEVER_Q3MAP2=/absolute/path/to/q3map2`.
